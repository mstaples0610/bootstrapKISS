The history of Bootstrap. Bootstrap is a free yet powerful front-end HTML, CSS, and JavaScript CSS framework. This open-source toolkit contains code dedicated to supporting the front-end web development process. It’s hosted and available for public use on GitHub. The framework was initially created by Mark Otto and Jacob Thornton to help maintain consistency across Twitter’s internal tools. During the development process, Bootstrap slowly evolved into something much larger and revolutionary for the web development industry. Once it went public in 2011, the rest is pretty much history.
